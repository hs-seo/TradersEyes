# Elliott Wave (엘리엇 파동) 분석 가이드

## 기본 원칙

엘리엇 파동 이론은 시장이 사회 심리의 반복적 패턴으로 움직인다는 관찰에 기반한다.
모든 추세는 **5개의 추진파(Motive Wave)** 와 **3개의 조정파(Corrective Wave)** 로 구성된다.

---

## 파동 구조

### 추진파 (Motive Wave) — 5파 구조
```
        3
       / \
      /   \
     /     4
    1      |
   / \     5
  /   2
```

| 파동 | 성격 | 특징 |
|------|------|------|
| Wave 1 | 추세 시작 | 약하고 의심받음. 종종 반등처럼 보임 |
| Wave 2 | 되돌림 | Wave 1의 50~61.8% 되돌림이 일반적 |
| Wave 3 | 가장 강한 파동 | 1.618~2.618 확장. 모멘텀, 거래량 최고 |
| Wave 4 | 조정 | Wave 1 가격대와 겹치지 않음. 종종 복잡 |
| Wave 5 | 마지막 추진 | 거래량 약화, 다이버전스 발생 |

### 조정파 (Corrective Wave) — 3파 구조 (A-B-C)
```
B
|\
| \
A  C
```

| 파동 | 성격 |
|------|------|
| Wave A | 첫 조정. 5파 구조로 하락 |
| Wave B | 되돌림. 불릿 트랩 발생 구간 |
| Wave C | 마지막 하락. 5파 구조로 이전 저점 갱신 |

---

## 3대 규칙 (절대 위반 불가)

1. **Rule 1**: Wave 2는 Wave 1의 시작점을 침범하지 않는다
2. **Rule 2**: Wave 3은 Wave 1, 5 중 가장 짧아서는 안 된다
3. **Rule 3**: Wave 4는 Wave 1의 가격대와 겹치지 않는다 (일반 시장 기준)

## 피보나치 레벨

### 되돌림 (Retracement)
| 파동 | 주요 레벨 |
|------|----------|
| Wave 2 | 0.382, 0.500, 0.618 |
| Wave 4 | 0.236, 0.382 (Wave 2보다 얕은 경우 많음) |
| Wave B | 0.500~0.786 |

### 확장 (Extension)
| 파동 | 주요 레벨 |
|------|----------|
| Wave 3 | 1.618, 2.000, 2.618 (Wave 1 대비) |
| Wave 5 | Wave 1과 같은 길이, 또는 1.618 |
| Wave C | Wave A와 동일, 또는 1.618 |

## 주요 패턴

### 조정 패턴 (Corrective Patterns)

#### Zigzag (5-3-5)
- A: 5파, B: 3파, C: 5파
- 가장 흔한 단순 조정
- B는 A의 75% 이상을 되돌리지 않음

#### Flat (3-3-5)
- A: 3파, B: 3파, C: 5파
- B가 A 시작점 근처까지 복귀
- Regular / Expanded / Running Flat

#### Triangle (3-3-3-3-3)
- 수렴하는 A-B-C-D-E 구조
- Wave 4 또는 Wave B에서 주로 발생
- 브레이크아웃 이후 급격한 추진 (Thrust)

#### WXY (Double Three) / WXYXZ (Triple Three)
- 복합 조정 패턴
- W, Y는 단순 조정 (Zigzag/Flat/Triangle)
- X는 연결파 (반대 방향 조정)

## ICT SMC와의 결합

### Wave 2 + Bullish OB
- Wave 2 되돌림이 Wave 1 기점의 OB에 닿을 때 최강 진입 구간
- FVG가 OB 내부에 존재하면 추가 신호

### Wave 3 진입 구조
1. Wave 2 조정 완료 확인 (0.5~0.618 피보)
2. CHoCH (Change of Character) — 저점 갱신 실패 후 고점 돌파
3. OB/FVG 미채움 구간 식별
4. 킬존(London/NY Open) 타이밍과 결합

### Wave 4 주의점
- Wave 4는 Wave 1 가격대 침범 시 카운팅 오류
- Flat 조정 시 False BOS 빈번 — 섣불리 숏 진입 금지
- 거래량 감소, 모멘텀 약화가 Wave 4 신호

### Wave 5 + 유동성 스윕
- Wave 5 정점에서 Sell-Side Liquidity 스윕 후 역전
- 다이버전스 + FVG 미채움 + Equal Highs = 고확률 숏 셋업
- Wave 5 = Wave 1 등치 구간에서 OB 확인

## 카운팅 실수 방지

| 실수 유형 | 확인 방법 |
|-----------|----------|
| Wave 3을 Wave 5로 오인 | Wave 3은 모멘텀/거래량 최고여야 함 |
| Flat의 Wave C 시작을 5파 Wave 1로 오인 | 직전 구조가 3파인지 5파인지 확인 |
| Triangle 내 파동 오카운팅 | 각 레그가 3파인지 검증 |
| 조정 완료 전 추진파 카운팅 | Rule 1~3 재확인 |
